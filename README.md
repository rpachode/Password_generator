# Password_generator

## Project Description
Password Generator is a Python application that creates strong and random passwords based on the length provided by the user. It uses random characters including letters, numbers, and special symbols.

## Technologies Used
- Python
- Random Module
- String Module

## Features
- Generates random passwords
- Allows users to choose password length
- Includes uppercase letters, lowercase letters, numbers, and symbols
- Creates secure and unique passwords

## How to Run the Project

1. Install Python.
2. Download or clone this repository.
3. Run the Python file:
    python Password_Generator.py
4. Enter the desired password length.

## Example Output

Enter the length for password: 10
Generated Password: aF7@kL92#m
